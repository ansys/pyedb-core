RaptorXAdvancedSettings#

class ansys.edb.core.simulation_setup.raptor_x_simulation_settings.RaptorXAdvancedSettings(sim_setup)#

Represents advanced settings for RaptorX simulations.

Methods

Attributes

RaptorXAdvancedSettings.advanced_options

dict { str : list`[:obj:`str] }: Advanced options.

RaptorXAdvancedSettings.auto_removal_sliver_poly

float: Automatic sliver polygon removal tolerance.

RaptorXAdvancedSettings.cells_per_wavelength

int: Number of cells that fit under each wavelength.

RaptorXAdvancedSettings.edge_mesh

str: Thickness and width of the exterior conductor filament.

RaptorXAdvancedSettings.eliminate_slit_per_holes

float: Threshold for strain or thermal relief slits and hole polygon areas.

RaptorXAdvancedSettings.mesh_frequency

str: Mesh frequency override.

RaptorXAdvancedSettings.msg

EDBObjMessage: Protobuf message that represents this object's ID.

RaptorXAdvancedSettings.net_settings_options

dict { str : list`[:obj:`str] }: Options for net settings.

RaptorXAdvancedSettings.override_shrink_factor

float: Shrink factor override value.

RaptorXAdvancedSettings.plane_projection_factor

float: Plane projection factor for reducing the mesh complexity of large metal planes.

RaptorXAdvancedSettings.use_accelerate_via_extraction

bool: Flag indicating if neighboring vias are simplified/merged.

RaptorXAdvancedSettings.use_auto_removal_sliver_poly

bool: Flag indicating if slight misaligned overlapping polygons are to be automatically aligned.

RaptorXAdvancedSettings.use_cells_per_wavelength

bool: Flag indicating if cells per wavelength are used.

RaptorXAdvancedSettings.use_edge_mesh

bool: Flag indicating if the edge mesh is used.

RaptorXAdvancedSettings.use_eliminate_slit_per_holes

bool: Flag indicating if strain relief or thermal relief slits and holes are removed.

RaptorXAdvancedSettings.use_enable_advanced_cap_effects

bool: Flag indicating if capacitance-related effects such as conformal dielectrics are applied.

RaptorXAdvancedSettings.use_enable_etch_transform

bool: Flag indicating if layout is "pre-distorted" based on foundry rules.

RaptorXAdvancedSettings.use_enable_hybrid_extraction

bool: Flag indicating if the modeler is to split the layout into two parts in an attempt to decrease the complexity.

RaptorXAdvancedSettings.use_enable_substrate_network_extraction

bool: Flag indicating if modeling of substrate coupling effects is enabled using equivalent distributed RC networks.

RaptorXAdvancedSettings.use_extract_floating_metals_dummy

bool: Flag indicating if floating metals are modeled as dummy fills.

RaptorXAdvancedSettings.use_extract_floating_metals_floating

bool: Flag indicating if floating metals are modeled as floating nets.

RaptorXAdvancedSettings.use_lde

bool: Flag indicating if variations in resistivity are taken into account.

RaptorXAdvancedSettings.use_mesh_frequency

bool: Flag indicating if the default meshing frequency is overridden.

RaptorXAdvancedSettings.use_override_shrink_factor

bool: Flag indicating if shrink factor is overridden.

RaptorXAdvancedSettings.use_plane_projection_factor

bool: Flag indicating if the plane projection factor is used.

RaptorXAdvancedSettings.use_relaxed_z_axis

bool: Flag indicating if simplified meshing is used along the z axis.
