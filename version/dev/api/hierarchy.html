Hierarchy#

Hierarchy objects are connectables that can act as a container for other connectables. Examples of hierarchy objects include subdesigns, MCAD components, and coordinate systems.

Object types#

cell_instance.CellInstance(msg)

Represents a cell instance object.

inst_array.InstArray(msg)

Represents an instance array object.

component_group.ComponentGroup(msg)

Represents a component group object.

group.Group(msg)

Represents a group object.

pin_group.PinGroup(msg)

Represents a pin group object.

model.Model(msg)

Represents a base hierarchy model object.

Represents a pin pair model object.

netlist_model.NetlistModel(msg)

sparameter_model.SParameterModel(msg)

Represents a SPICE model object.

Represents a 3D structure.

Represents a via group object.
